Вот инструкция по установке и использованию приложения "Связаться с нами":
1. Распаковать архив и поместить файлы приложения в папку веб-сервера (например, htdocs).
2. Импортировать базу данных из файла 127_0_0_1.sql, которая находится в папке backup.
3. Отредактировать файл config.php которая находится в папке config, указав параметры подключения к базе данных. 
	Пример:
	$servername = "localhost";  —————> название вашего сервера
	$username = "root";	    —————> имя юзера даты базы
	$password = "";		    —————> пароль юзера даты базы 
	$dbname = "contact_us";	    —————> название даты базы должно оставаться таким же
4. Убедиться, что на сервере установлены необходимые библиотеки:
	PHP >= 7.0
	MySQL
	Bootstrap 5
	Библиотека для работы с БД (MYSQLI)
5. При отправке данных, они также отправляются на электронную почту администратора. Вы можете зайти в мой аккаунт гугл и проверить данные либо изменить почту на свою.

Использование!
1.Открыть в браузере главную страницу приложения.
2.Заполнить форму и отправить данные. Проверка на заполнение обязательных полей происходит на клиенте.
3.Данные отправляются на скрипт result.php, который сохраняет их в БД и отправляет письмо администратору.
4.Посмотреть список отправленных данных можно на странице all_results.php. Реализована постраничная навигация.
5.Администратор может удалять и редактировать записи в БД.
6.Постраничная навигация: В странице результатов есть постраничная навигация. Если данных в одной странице больше 10, то остальные данные переходят на другую страницу.
